# Sneha Sammilana Foundation Website

A modern, production-ready website for the **Sneha Sammilana Foundation**, built with **Next.js 14** to replace a legacy WordPress installation. The project focuses on performance, maintainability, accessibility, and a zero-cost content management workflow capable of serving hundreds of megabytes of event media without traditional cloud storage.

---

## ✨ Features

- ⚡ Next.js 14 App Router
- 🎨 Responsive UI built from Stitch designs
- 📱 Mobile-first experience
- 🦴 Route-level skeleton loading
- 🎭 Subtle Framer Motion animations
- 🖼️ Automatic event discovery
- 🌍 CDN-backed image delivery
- 📸 Progressive event gallery
- ♿ Accessible semantic components
- 🚀 Optimized for Vercel deployment
